## Key Takeaways

1. **API routes run on the server** - They have access to `process.env`
2. **Environment variables are server-side only** - Not exposed to browser
3. **Frontend makes API calls** - Using relative paths like `/api/parse-resume`
4. **Backend uses Gemini API** - With the API key from environment variables
5. **Deployment platforms inject env vars** - At build/runtime
